# เรื่องบ้านบ้านน็อคดาวน์ นครสวรรค์ – Website Design System

---

## 1. Brand Overview

เรื่องบ้านบ้านน็อคดาวน์ นครสวรรค์ คือธุรกิจรับสร้างบ้านน็อคดาวน์ครบวงจร  
เน้นความ “คุ้มค่า รวดเร็ว และใช้งานได้จริง”

ภาพลักษณ์แบรนด์ควรให้ความรู้สึก:
- Modern Minimal
- น่าเชื่อถือ (Trustworthy)
- เข้าถึงง่าย (Affordable)
- งานก่อสร้างที่ดู “สะอาด เป็นระบบ”
- บ้านพร้อมอยู่ (Ready-to-live lifestyle)

---

## 2. Logo Usage

### Primary Logo
- โลโก้บ้าน + ตัวอักษรสีเขียว + ไฮไลท์สีเหลือง
- ใช้เวอร์ชันพื้นขาวเป็นหลัก

### Logo Rules
- เว้นระยะรอบโลโก้ = ความสูงของตัวอักษร
- ห้ามยืด / เปลี่ยนสี
- ใช้ icon บ้านอย่างเดียวได้ (favicon / mobile)

### Recommended Use
- Navbar → ใช้ icon + text (ขนาดเล็ก)
- Hero → ใช้ full logo
- Mobile → ใช้ icon only

---

## 3. Color System

### Primary Colors (Brand Core)

- **Deep Green (Main Brand)**
  - HEX: #0F6B3A
  - Usage: Logo, headings, CTA background

- **Golden Yellow (Highlight)**
  - HEX: #F4C21D
  - Usage: Accent text, highlights, CTA hover

---

### Secondary Colors

- **Forest Green**
  - HEX: #1E8A4D
  - Usage: Icons, secondary buttons

- **Light Green Tint**
  - HEX: #E8F5EC
  - Usage: Section background

- **White**
  - HEX: #FFFFFF
  - Usage: Main background

- **Dark Text**
  - HEX: #1A1A1A
  - Usage: Body text

---

### Functional Colors

- Success: #28A745  
- Warning: #FFC107  
- Error: #DC3545  

---

## 4. Typography

### Headings
- Font: **Prompt / Kanit / Sarabun**
- Style: Bold / Semi-bold
- Tone: Clean, modern Thai

### Body Text
- Font: **Prompt / Sarabun**
- Weight: Regular

### Hierarchy

- H1: ใหญ่ + Bold  
- H2: Semi-bold  
- H3: Medium  
- Body: Regular  
- Caption: Light  

---

## 5. UI Components

### Buttons

#### Primary Button
- Background: Green (#0F6B3A)
- Text: White
- Radius: 10–16px
- Hover: Darker green + slight lift

#### Secondary Button
- Border: Green
- Text: Green
- Background: Transparent

---

### Cards
- Background: White
- Shadow: Soft (0 10px 30px rgba(0,0,0,0.05))
- Radius: 16px
- Used for: services, portfolio

---

### Icons
- Style: Line + minimal fill
- Color: Green / Yellow accent
- Theme: บ้าน / ก่อสร้าง / โมเดิร์น

---

## 6. Layout Style

Inspired by reference website:

- Large hero image
- Rounded containers (modern UI)
- Grid-based layout
- Clean spacing
- Section separation using background tint

### Structure Pattern

- Hero (image + text left)
- Services (card grid)
- Portfolio (image grid)
- About / stats
- CTA
- Contact

## 7. Imagery Direction

## 11. Do’s & Don’ts

### Do:
✔ ใช้สีเขียว + เหลืองเป็นหลัก  
✔ ใช้ layout สะอาด  
✔ เน้นภาพบ้านจริง  

### Don’t:
✘ ใช้หลายสีเกินไป  
✘ ใส่ข้อมูลแน่นเกิน  
✘ ใช้ฟอนต์แฟนซี
