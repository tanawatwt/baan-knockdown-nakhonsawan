# Website Structure – เรื่องบ้านบ้านน็อคดาวน์ นครสวรรค์

---

## 1. Website Goal

เว็บไซต์นี้มีเป้าหมายเพื่อสร้างความน่าเชื่อถือให้กับธุรกิจรับสร้างบ้านน็อคดาวน์ในจังหวัดนครสวรรค์และพื้นที่ใกล้เคียง พร้อมช่วยให้ลูกค้าสามารถดูบริการ ดูผลงาน ดูราคาเบื้องต้น และติดต่อขอใบเสนอราคาได้ง่าย

Primary Goal:
- ให้ลูกค้าติดต่อสอบถาม / ขอใบเสนอราคา
- เพิ่มความน่าเชื่อถือด้วยภาพผลงานจริงและข้อมูลบริการครบถ้วน
- อธิบายจุดเด่นของบ้านน็อคดาวน์: สร้างไว ราคาคุ้มค่า ปรับแบบได้

## 4. Home Page

### 4.1 Hero Section

Content:
- Main headline: รับสร้างบ้านน็อคดาวน์ครบวงจร ราคาคุ้มค่า สร้างไว
- Supporting text: ออกแบบ ผลิต และติดตั้งครบจบในที่เดียว พร้อมให้คำปรึกษาฟรี ปรับแบบได้ตามงบประมาณและพื้นที่ของลูกค้า
- Primary CTA: ขอใบเสนอราคา
- Secondary CTA: ดูแบบบ้าน

- สร้างเสร็จไว 15–45 วัน

## 8. Process Page

### Page Header
- Title: ขั้นตอนการทำงาน
- Intro: วางแผนชัดเจน ตั้งแต่เริ่มต้นจนส่งมอบงาน

---

### Process Steps

1. ติดต่อสอบถาม / ขอใบเสนอราคา
   - ลูกค้าส่งรายละเอียด งบประมาณ หรือแบบที่สนใจ

2. ประเมินหน้างาน / สรุปแบบ
   - ทีมงานประเมินพื้นที่และสรุปความต้องการ

3. ตกลงราคาและมัดจำ
   - ยืนยันแบบ ราคา และเงื่อนไขการทำงาน

4. เริ่มผลิตโครงสร้าง
   - ผลิตตามแบบและวัสดุที่ตกลง

5. ติดตั้งหน้างาน
   - ทีมงานเข้าติดตั้งตามกำหนด

6. ส่งมอบงาน
   - ตรวจสอบงานและส่งมอบให้ลูกค้า

## 13. UX Guidelines

- Mobile-first layout
- CTA visible above the fold
- Sticky call / LINE buttons on mobile
- Use real project photos where possible
- Keep content short and easy to scan
- Show price as “เริ่มต้น” to avoid fixed quotation confusion
- Use clear section headings
- Optimize images for fast loading
